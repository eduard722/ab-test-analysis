# 📦 A/B Test Analysis

## 📌 Описание проекта

Провёл анализ A/B тестов в приложении доставки для оценки влияния продуктовых изменений на пользовательскую активность.

---

## 🎯 Что сделано:

- Проверка предпосылок (тест Левена)
- ANOVA для сравнения групп
- Post-hoc анализ (Tukey HSD)
- Двухфакторный анализ (group × segment)
- Анализ interaction effect

---

## 📊 Данные

### 5_task_1:
- id – id клиента в эксперименте
- group – в каком разрешении показывались картинки (A – прямоугольные 16:9, B – квадратные, C – прямоугольные 12:4)
- events – сколько блюд суммарно было заказано за период

### 5_task_2:
- id – id клиента в эксперименте
- segment – сегмент (high/low)
- group – вид кнопки (control – старая версия, test – новая версия)

---

## 📈 Результаты
- Выявлены статистически значимые различия между группами
- Обнаружено влияние сегмента пользователей на эффект
- Новая версия кнопки увеличивает метрику во всех сегментах → рекомендовано внедрение
